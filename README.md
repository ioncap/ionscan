# IonScan - Network Intelligence Suite

IonScan is a modular bash-based script for network discovery, vulnerability scanning, and reporting. It is designed for system administrators and security professionals to audit and monitor their networks.

## Features

*   **Modular Architecture:** Easily extendable with new scripts and tools.
*   **Network Reconnaissance:** Passive and active scanning techniques.
*   **Vulnerability Scanning:** Integrated with Nmap's script engine (NSE) and Vulners.
*   **Exploitation Helpers:** Modules for assisting in penetration testing tasks.
*   **Reporting:** Generates an HTML dashboard of the findings.
